css优先级如下：
通用选择器（*） < 元素(类型)选择器 < 类选择器 < 属性选择器 < 伪类 < ID 选择器 <内联样式

css内部是按照每条样式的权重值来计算优先级的，各类型选择器所对应的权重值：
元素, 伪元素: 1 C (0,0,0,1)     伪元素：：first-letter  :first-line  :before  :after
类, 伪类, 属性: 1 C (0,0,1,0)    一个class就是一个类名
ID: 1 C (0,1,0,0) 
内联样式: 1 C (1,0,0,0)



每条样式的权重值就是该样式所包含的所有选择器相对应的权重值之和：
.div p    11
div p    2


强制使用样式：!important
